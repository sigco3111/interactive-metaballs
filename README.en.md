# 💧 interactive-metaballs

> HTML5 Canvas + SDF-based metaball liquid-merging simulation — drag blobs with your mouse, watch them blend like liquid metal

Multiple blobs rendered via signed-distance-field (SDF) composition on HTML5 Canvas 2D produce a fluid metaball effect: nearby blobs merge smoothly, distant ones split apart. Move your mouse to attract the nearest blob; click to inject a new one. Nearest-neighbor calculation runs in a Web Worker so the main thread stays responsive, keeping 60fps stable.
